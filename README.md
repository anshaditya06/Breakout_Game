# Breakout Game

A simple Breakout-style game built with Python and Turtle.

## Features
- Move the paddle left and right
- Bounce the ball to break bricks
- Keep score and play until the ball is lost

## Requirements
- Python 3
- Turtle (included with standard Python)

## How to Run
From the project directory, run:

```bash
python3 main.py
```

## Project Files
- `main.py` - game setup and main loop
- `ball.py` - ball movement and collisions
- `paddle.py` - paddle controls
- `brick.py` - brick behavior
- `scoreboard.py` - score display
